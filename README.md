# Convertil

## Sobre o Projeto

O **Convertil** foi uma plataforma web desenvolvida para centralizar diversas ferramentas online em um único lugar, oferecendo praticidade para usuários que precisam realizar conversões, cálculos e manipulação de dados diretamente pelo navegador.

O projeto foi desenvolvido integralmente utilizando tecnologias web tradicionais, com foco em desempenho, simplicidade de uso e compatibilidade com diferentes dispositivos.

## Tecnologias Utilizadas

### Front-end
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Back-end
- Node.js

## Objetivos

O principal objetivo do Convertil foi disponibilizar ferramentas úteis de forma rápida e acessível, eliminando a necessidade de instalar programas específicos para tarefas simples do dia a dia.

Entre os benefícios buscados estavam:

- Interface simples e intuitiva.
- Execução rápida das operações.
- Compatibilidade com dispositivos móveis e desktops.
- Centralização de diversas utilidades em uma única plataforma.
- Facilidade de manutenção e expansão.

## Arquitetura

O sistema foi dividido em duas camadas principais:

### Cliente (Front-end)

Responsável por:

- Renderização da interface.
- Validação inicial dos dados.
- Interação do usuário.
- Comunicação com a API.

### Servidor (Back-end)

Responsável por:

- Processamento das requisições.
- Execução das conversões e cálculos.
- Validação dos dados recebidos.
- Retorno dos resultados ao cliente.

## Funcionalidades

Dependendo da versão do projeto, o Convertil disponibilizava ferramentas como:

- Conversores de formatos.
- Calculadoras online.
- Formatadores de texto.
- Ferramentas para desenvolvedores.
- Utilitários para manipulação de dados.
- Conversões diversas para facilitar tarefas cotidianas.

## Desafios do Desenvolvimento

Durante o desenvolvimento, foram trabalhados conceitos como:

- Manipulação do DOM.
- Tratamento de eventos.
- Comunicação assíncrona.
- Organização de código JavaScript.
- Estruturação de APIs em Node.js.
- Tratamento e validação de dados.
- Responsividade da interface.

## Aprendizados

O desenvolvimento do Convertil contribuiu para o aprimoramento de conhecimentos em:

- Desenvolvimento Front-end.
- Desenvolvimento Back-end.
- Arquitetura de aplicações web.
- Integração entre cliente e servidor.

- Publicação e manutenção de aplicações web.

## Status do Projeto

Atualmente o projeto encontra-se descontinuado devido à perda do domínio utilizado para hospedagem pública. Entretanto, o desenvolvimento do Convertil representou uma experiência significativa na construção de aplicações web completas utilizando HTML, CSS, JavaScript e Node.js.
