Perguntas Frequentes

O VoxText armazena minhas gravações?

Não! Todas as gravações são processadas localmente no seu navegador e nunca são enviadas para nossos servidores. Sua privacidade é nossa prioridade.

Quais navegadores são compatíveis?

O VoxText funciona na maioria dos navegadores modernos como Chrome, Firefox, Edge e Safari. Para melhor desempenho, recomendamos usar a versão mais recente do seu navegador.

Posso usar o VoxText offline?

Atualmente, o VoxText requer conexão com a internet para carregar os modelos de reconhecimento de voz. No entanto, uma vez carregados, você pode usar o microfone mesmo em conexões instáveis.
